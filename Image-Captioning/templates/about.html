About


Image Caption through AI

Image Captioning is the process of generating a textual description of an image. It is a challenging problem in computer vision and natural language processing. The goal of this project is to build a deep learning model that can generate a caption for an image. The model will be trained on the Flickr8k dataset, which contains 8,000 images that are each paired with five different captions.
